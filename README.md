# pollinator-insects-jaipur
# Pollinator Insects of Jaipur Region

This repository provides a basic list and study of pollinator insects found in Jaipur region, Rajasthan.

## Major Pollinator Groups

### Bees (Hymenoptera)
- Apis dorsata (Rock Bee)
- Apis cerana indica (Indian Honey Bee)
- Apis florea (Little Bee)
- Trigona spp. (Stingless Bees)

### Butterflies (Lepidoptera)
- Papilio demoleus (Lime Butterfly)
- Danaus chrysippus (Plain Tiger)
- Pieris brassicae (Cabbage Butterfly)

### Flies (Diptera)
- Syrphid flies (Hoverflies)

### Beetles (Coleoptera)
- Scarab beetles (Flower visitors)

## Importance
Pollinator insects play a crucial role in crop production and biodiversity conservation in semi-arid regions like Jaipur.

## Future Scope
More species records, seasonal observations, and field-based data will be added regularly.
